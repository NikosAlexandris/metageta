Transform(inxmlstring, transform, outxmlfile)

source code 

Transform a metadata record to XML using an XSL stylesheet

Parameters:
  • inxmlstring - metadata record in XML format.
  • transform - may be one of the pre-defined XSL transforms or a path to a custom XSL file.
  • outxmlfile - File to write transformed metadata to.

CreateMEF(outdir, xmlfile, uid, overviews=[])

source code 

Generate Geonetwork "Metadata Exchange Format" from an ISO19139 XML record

Parameters:
  • outdir - Directory to write MEF file to.
  • xmlfile - XML file to create MEF from.
  • uid - ID of metadata record (UUID/GUID string).
  • overviews - List of overview image file (e.g quicklooks & thumbnails) OPTIONAL.
