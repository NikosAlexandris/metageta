Module runcrawler

source code

Script to run the Metadata Crawler

Contains code to show GUI to gather input arguments when none are provided To run, call the eponymous batch file which sets the required environment variables

Usage:

   runcrawler.bat -d dir -x xls -s shp -l log {-o} {--nomd} {--gui} {--debug}
Arguments:
  • -d [dir]: Directory to to recursively search for imagery
  • -x [xls]: MS Excel spreadsheet to wrtite metadata to
  • -s [shp]: ESRI Shapefile to write extents to
  • -l [log]: Log file to write messages to
  • -o : Generate overview (quicklook/thumbnail) images")
  • --nomd : Extract metadata (crawl), if False just get basic file info (walk)")
  • --gui : Show the GUI progress dialog")
  • --debug : Turn debug output on
