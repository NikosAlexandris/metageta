Script to run the Metadata Crawler

Contains code to show GUI to gather input arguments when none are provided To run, call the eponymous batch file which sets the required environment variables

Usage:

   runcrawler.bat dir xls shp log
Arguments:
  • dir: Directory to to recursively search for imagery
  • xls: MS Excel spreadsheet to wrtite metadata to
  • shp: ESRI Shapefile to write extents to
  • log: Log file to write messages to
