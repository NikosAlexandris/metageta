Module runtransform

source code

Script to run the Metadata Transforms

Contains code to show GUI to gather input arguments when none are provided To run, call the eponymous batch file which sets the required environment variables

Usage:

   runtransform.bat xls xsl [dir]
Arguments:
  • xls: MS Excel spreadsheet to read from
  • xsl: XSL transform - may be one of the pre-defined XSL transforms or a path to a custom XSL file.
  • dir: Optional - directory to write XML files to. If not supplied, XML files are written to the directory containing the image
